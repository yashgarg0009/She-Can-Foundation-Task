# She Can Foundation Website

This is a simple and responsive NGO website created for the internship task of She Can Foundation.

## Features

- Responsive Design
- Dark Mode Toggle
- Modern Landing Page
- About Section
- Footer Section
- Simple HTML, CSS and JavaScript

## Technologies Used

- HTML
- CSS
- JavaScript

## Purpose

The purpose of this project is to create a clean and user-friendly webpage for an NGO that works towards women empowerment and community support.
